play_circle使用指南

按照以下步骤快速开始使用本项目：

1

准备数据集

将数据集整理为 .csv 或其他可读格式，并确保数据集路径正确。数据集应包含特征列和目标变量列。

2

数据预处理

使用 encode.py 对分类变量进行编码，使用 tools.py 中的函数进行数据标准化和特征分析。可选运行 corr.py 分析特征相关性。

3

运行单个模型

直接运行对应的算法文件，如 LightGBM-optuna.py，该文件将使用 Optuna 自动调参并保存最优模型到 savemodel/ 目录。

4

批量运行所有模型

运行 runall.py 可以依次执行所有算法，自动进行参数优化和模型训练，结果统一保存便于对比。

5

分析结果

使用 paras_read.py 提取最优参数，使用 tools.py 中的函数进行可视化分析，包括回归拟合图、残差图、特征重要性图等。

tips_and_updates提示

建议优先使用 uv 进行环境管理，可以快速安装所有依赖并创建独立的虚拟环境。

download环境安装

本项目需要以下 Python 依赖库：

使用 pip 安装

pip install optuna scikit-learn pandas numpy matplotlib seaborn lightgbm catboost shap xgboost

使用 uv 安装（推荐）

项目已包含 pyproject.toml 和 uv.lock 文件，可以使用 uv 快速设置环境：

# 安装 uv（如果尚未安装） pip install uv # 同步项目依赖 uv sync # 激活虚拟环境 uv shell

主要依赖说明

  • optuna - 自动化超参数优化框架
  • scikit-learn - 机器学习基础库
  • lightgbm - LightGBM 梯度提升库
  • catboost - CatBoost 模型库
  • shap - 模型可解释性分析库
  • xgboost - XGBoost 梯度提升库
  • pandas, numpy - 数据处理基础库

Python 版本要求

建议使用 Python 3.8 或更高版本以获得最佳兼容性和性能。
